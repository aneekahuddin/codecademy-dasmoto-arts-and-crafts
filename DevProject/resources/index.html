HTML/CSS Cheat Sheet

Introduction to HTML

  • HTML stands for HyperText Markup Language and is used to create the structure and content of a webpage.
  • CSS can change the look of HTML elements. In order to do this, CSS must select HTML elements, then apply styles to them.
  • JS is a programming language that powers the dynamic behavior on most websites. There are seven data types: numbers, strings, boolean, floats, undefined, null, object.


HTML Element Description Common Attributes
!DOCTYPE html tells the browser what version of HTML to run
html should contain all HTML code
head info about the webpage: title, css etc. belong here
title creates the webpage title, appears in browser tab
header defines the header of the visible content, usually the navigation is included in here
nav creates the navigation bar
main this is the "body" of the body. displays the main part of the page
section holds content into a categorization such as articles, blogs, comments etc
aside has info that is related to the main content but not required for understanding
figure encapsulates media and is often used with figcaption (to caption the media)
body Visible content should be placed within the opening and closing tags.
span or div specifies a block of code. often used to apply attributes to blocks of code or assign an id/class style, id, class
br creates a line break
!--- comment -- creates a comment
h1 to h6 Headings/subheadings. These enlarge titles. style, id, class
p this creates a text block style, id, class
em or strong italicize or bold, respectively
ol and ul ordered (numbered) list and unordered(bulleted) list
a anchor tag, used to link to internal, external or same page href*
img used to display an image src*, alt
video used to display video src*, controls*, alt, width, height
audio used to display audio, often used with source tag controls, src (unless source tag is used with type attribute)
embed used to embed media from external sources src
tr creates table row
td creates table data/cell
th creates table heading in that table cell
thead designates the head of the table
tbody designates the body of the table
tfooter designates the footer of the table
form allow users to input information and send it action*, method*
input allows users to input information. if you set the type attribute equal to:
  • "text" - single row field for text input
  • "password" - single row field that censors input
  • "number" - single row field for number input
  • "range" - creates a slider to select from number range. used with min and max attributes
  • "checkbox" - creates a single checkbox (multiple options)
  • "radio" - creates a single radio (one option only)
  • "list" - will pair to datalist element instead of form
  • "submit" - will create a submit button
the first six pair to form 		type*, name, value, id
select creates a dropdown list populated from multiple option tags nested under id
datalist creates a searchable dropdown list from multiple option tags, works with input id
option creates an option in select or datalist value*
textarea creates a text-box multi-line input rows, cols
label creates a label for the form question, can be matched by using ID for

*These are mandatory attributes.



Attribute Description
style add CSS style to the element
id most specific way of identifying an element besides important
class a specific way to identify an element. if the element is identified by an ID, that code will take precedence over class code.
!important this is not an attribute but a flag. it will override any specificity of style
src links to the source. be sure to use "" around the source
alt displays alternative text if media cannot be displayed. use quotes
controls adds video controls to video, such as play, pause, volume
width and height gives media a width and a height
href include link in quotes, if linking ID use a hashtag
colspan allows a data cell to span multiple columns, set equal to a #
rowspan allows a data cell to span multiple rows, set equal to a #
action links to where information (from a form, etc.) goes
method determines how the information is sent and processed
name the field that accepts input
value the input value, assigned to name
list calls out a list
for helps to match an element to another by matching for with an id
rows amd cols creates the rows and columns in a textbox area
type can define what type of form, can define what a link may be (CSS stylesheet)
min or max to create a minimum and maximum, often used for range forms
step creates the amount of increments on a range slider on a form
required makes a form required
maxlength or minlength creates a max/min number of characters in a text field
pattern creates a set of restriction and makes certain that input falls into pattern (i.e. uppercase)

font-weight how thin or thick text is displayed bold or 100
text-align places text in the left, right or center right, left, or center
color color of text seafoamgreen
background-color color behind the text blue
opacity sets opacity of an obj 0 (transparent) to 1
background-image sets the background of an element to an image url("xyz.com")
